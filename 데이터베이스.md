# 데이터 베이스

1. Key

   > Key ? 검색, 정렬시 tuple을 구분할 수 있는 기준이 되는 특성

   * Primary Key (기본키)
     * 후보키 중 선택한 main key
     * null 값을 가질 수 없음
     * 동일한 값이 중복될 수 없음
   * Candidate Key (후보키)
     * Tuple을 유일하게 식별하기 위해 사용하는 속성들의 부분 집합 (기본키로 사용할 수 있는 속성들)
     * 유일성 - Key로 하나의 Tuple을 유일하게 식별할 수 있음
     * 최소성 - 꼭 필요한 속성으로만 구성
   * Alternate Key (대체키)
     * 후보키 중 기본키를 제외한 나머지 키 = 보조키
   * Super Key (슈퍼키)
     * 유일성은 만족하지만, 최소성은 만족하지 못하는 키
